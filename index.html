Portfolio

UDCIS Scheduler

ReactTS website made to help UD Computer Science students plan their college career. Students can create multiple plans where they can add and structure classes across each semester. Made in a group using Agile framework.

Serial Port Monitor

Windows app to monitor a RS232 data stream from a serial port device. The user can enter a threshold and is notified if the data ever changes by more than that threshold. Originally built to be used when surveying the seabed and measuring depth.

Analysis of Radiolaria

NIH sponsored research lead by Dr. John Jungck at the University of Delaware. I studied several Radiolaria samples and discovered a pattern in their structure that may assist in their identification. I presented my findings using VR and 3D projection.

UNIX Shell

A basic UNIX shell written in C. that is able to run UNIX built in commands (ls, cd, pwd etc.), and user programs.
